ESLint, Prettier e GitHooks com Husky em React.js

Aprenda a validar seu código antes do commit no Git usando GitHooks com Husky, ESLint e Prettier em projetos React.js

alt="Banner do post com ícone das ferramentas" />

Neste post quero compartilhar como configurar o ESLint e Prettier em projetos Reactjs. Validar as regras usando os Githooks com Husky. Este setup vai garantir que, antes de fazer um commit, nosso código seja testado. Dessa forma, evitando que o código ruim ou mal formatado seja enviado em um pull-request para nosso repositório Git.

O foco aqui será o setup do ESLint, Prettier e Husky em uma aplicação Reactjs. Mas, os mesmos princípios podem ser aplicados em outros projetos JavaScript, uma API Nodejs, como exemplo.

ESLint

Se você está começando agora a trabalhar com desenvolvimento de projetos com Reactjs. No início, essa flexibilidade é ótima porque conseguimos desenvolver com agilidade, porém a longo prazo isso se torna um grande problema.
